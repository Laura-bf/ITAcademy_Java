# M1 Variables

![Java](https://img.shields.io/badge/Java-8-ED8B00?logo=openjdk)

## Description

A console application demonstrating variable declaration, type casting, and array reversal. The project is structured in three progressive levels covering string and integer operations, leap year calculations, type conversion, and array manipulation.

## Technologies

- Java 8

## Key Concepts

- Variable declaration and initialization (String, int, double, float)
- Type casting (double to int, float, String)
- Leap year calculation using loops
- Array creation and reversal
- Formatted console output

## Project Structure

```
M1_Variables/
  src/
    N1/
      Fase1.java
      Fase2.java
      Fase3.java
      Fase4.java
    N2/
      N2.java
    N3/
      N3.java
```

## How to Run

Each file contains its own `main` method and can be run independently.

```bash
cd src
javac N1/Fase1.java
java N1.Fase1
```

Replace `N1.Fase1` with another class (e.g. `N2.N2`, `N3.N3`) to run a different level.
