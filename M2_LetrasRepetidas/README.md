# M2 Letras Repetidas

![Java](https://img.shields.io/badge/Java-11%2B-ED8B00?logo=openjdk)

## Description

A console application that iterates over character arrays, classifies letters as vowels or consonants, counts character frequency using HashMaps, builds inverted asterisk pyramids and number staircases, and simulates a digital stopwatch timer.

## Technologies

- Java 11+

## Key Concepts

- Char array and ArrayList iteration
- Character classification (vowel, consonant, digit)
- HashMap frequency counting
- Nested loops for pyramid and number patterns
- Thread.sleep() for timed execution
- DecimalFormat for time formatting

## How to Run

Each file contains its own `main` method and can be run independently.

```bash
cd src
javac N1/Fase1.java
java N1.Fase1
```

Replace `N1.Fase1` with another class (e.g. `N2.PiramideAsteriscos`, `N3.Cronometro`) to run a different exercise.
