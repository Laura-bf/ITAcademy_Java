# M3 Nombres Ciudades

![Java](https://img.shields.io/badge/Java-11%2B-ED8B00?logo=openjdk)

## Description

A console application that reads city names via scanner input, sorts and manipulates them in arrays, replaces characters, reverses strings, calculates student grade averages from a 2D array, and generates the Fibonacci sequence up to a given number.

## Technologies

- Java 11+

## Key Concepts

- Scanner input and string handling
- ArrayList and Collections.sort()
- String character replacement
- Char array creation and string reversal
- 2D arrays for grade storage and average calculation
- Fibonacci sequence generation

## How to Run

Each file contains its own `main` method and can be run independently.

```bash
cd src
javac N1/Fase1.java
java N1.Fase1
```

Replace `N1.Fase1` with another class (e.g. `N2.MediaAlumnos`, `N3.Fibonacci`) to run a different exercise.
